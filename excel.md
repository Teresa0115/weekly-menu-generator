# 台灣小公司 Excel 記帳系統

**檔案名稱**：`nosif_accounting_2025.xlsx`  
**建立日期**：2026-04-28  
**科目數量**：224 個科目  
**支援功能**：日記簿、試算表、資產負債表、損益表、VAT計算

---

## 📋 系統概述

這是一個完整的台灣會計記帳系統，用 Excel 實現：
- 自動讀取 224 個科目（來自捷瑞會計系統）
- 只需在「日記簿」輸入傳票
- 所有報表自動計算（試算表、BS、IS）
- 支援台灣增值稅（VAT）計算

## 🗂️ 工作表說明

### 1. 說明
- 系統使用說明
- 科目表修改方式
- 日記簿輸入欄位說明

### 2. 科目表
- **欄位**：科目代號、科目名稱、借/貸、科目類別
- **功能**：顯示所有 224 個科目
- **可修改**：用戶可新增或刪除科目

**科目分類規則**：
- 1xxx = 資產
- 2xxx = 負債
- 3xxx = 股東權益
- 4xxx = 收入
- 5xxx = 銷貨成本
- 6xxx = 費用
- 7xxx = 非營業收入
- 8xxx = 非營業費用
- 9xxx = 所得稅

### 3. 日記簿（主要輸入介面）
**欄位**：
| 欄位 | 說明 | 格式 |
|------|------|------|
| 傳票號 | 唯一識別號 | 例如：2401001 |
| 日期 | 交易日期 | YYYY/MM/DD |
| 摘要 | 交易說明 | 文字 |
| 科目代號 | 必須在科目表中 | 4 位數字 |
| 科目名稱 | 自動帶入（可編輯） | 文字 |
| 借方金額 | 借方記錄 | 數字（無千位符） |
| 貸方金額 | 貸方記錄 | 數字（無千位符） |

**規則**：
- 同一傳票號的所有行記錄同一筆交易
- 借方 = 貸方（雙重記帳原則）
- 同一行只能填借方或貸方（不能同時填兩個）

**範例**：
```
傳票號: 2401001
日期: 2025/01/03
摘要: 公司創立

第 1 行：科目 11121（銀行存款）借方 1,000,000
第 2 行：科目 3110（實收資本）貸方 1,000,000
```

### 4. 試算表
- **自動生成**：從日記簿統計
- **欄位**：科目代號、科目名稱、借方合計、貸方合計、餘額
- **公式**：`=SUMIFS(日記簿[借方], 日記簿[科目代號], [代號])`

**最後一行**：
- 合計行（借方合計 = 貸方合計）
- 借貸平衡檢查（✓ 平衡 或 ✗ 不平衡）

### 5. 資產負債表（BS）
- **自動生成**：從試算表彙總
- **左側**：資產（流動資產 + 非流動資產）
- **右側**：負債及權益（負債 + 股東權益）
- **驗證**：資產總計 = 負債及權益總計

**公式**：
```
流動資產 = SUMIFS(試算表[餘額], 試算表[類別], "資產")
負債 = SUMIFS(試算表[餘額], 試算表[類別], "負債")
權益 = SUMIFS(試算表[餘額], 試算表[類別], "權益")
```

### 6. 損益表（IS）
- **自動生成**：從試算表彙總
- **結構**：
  ```
  營業收入（4xxx）
  - 銷貨成本（5xxx）
  = 營業毛利
  - 營業費用（6xxx）
  = 營業淨利
  + 非營業收入（7xxx）
  - 非營業費用（8xxx）
  = 稅前損益
  - 所得稅費用（9xxx）
  = 稅後損益
  ```

### 7. VAT 計算
- **進項稅額**：科目 1144 餘額
- **銷項稅額**：科目 2171 餘額
- **應繳 VAT**：銷項稅 - 進項稅

**說明**：
- 應繳 VAT > 0：需繳稅
- 應繳 VAT < 0：可退稅（待退金額）
- 應繳 VAT = 0：不需繳退

**台灣營業稅率**：5%

## 📝 使用流程

### 步驟 1：準備科目
- 系統已包含 224 個標準科目
- 如需修改，在「科目表」編輯

### 步驟 2：輸入傳票
在「日記簿」逐行輸入每筆交易：
1. 填傳票號（同一筆交易用同一號）
2. 填日期
3. 填摘要
4. 選科目代號
5. 填借或貸金額

### 步驟 3：檢查平衡
- 「試算表」最後會顯示 ✓ 平衡
- 若顯示 ✗ 不平衡，檢查日記簿

### 步驟 4：查看報表
- 「資產負債表」自動生成
- 「損益表」自動生成
- 「VAT計算」自動生成

## 💡 常見操作

### 新增科目
1. 在「科目表」底部新增一行
2. 填入：代號（4位數）、名稱、借/貸
3. 日記簿和所有報表自動應用

### 新增傳票
1. 在「日記簿」最後插入新行
2. 填入傳票資訊
3. 試算表自動更新

### 查詢特定科目
1. 「試算表」可用 Ctrl+F 搜尋代號
2. 看到該科目的借貸合計和餘額

### 列印報表
- 分別列印「資產負債表」和「損益表」工作表
- 建議設定頁面格式為橫向

## ⚠️ 重要提醒

### 公式保護
- 「試算表」、「資產負債表」、「損益表」包含公式
- **不要刪除列或欄**（會破壞公式）
- 若要修改，只在「科目表」和「日記簿」操作

### 科目代號
- 必須是 4 位數字（1000～9999）
- 代號唯一性（不要重複）
- 建議保留原有代號結構（1xxx 資產、2xxx 負債等）

### 日期格式
- 使用 `YYYY/MM/DD` 格式
- 例如：2025/01/03、2025/12/31

### 金額輸入
- 只輸入數字（不含逗號或符號）
- 例如：1000000（不是 1,000,000 或 $1,000,000）
- 系統會自動格式化顯示千位符
