### ⏳ 第 2 階段：商品導入（待開始）

#### 2.1 從菜單生成器導出商品

**來源**：https://weekly-menu-generator-production.up.railway.app

**商品清單範例**：
```json
{
  "categories": [
    {
      "name": "鹹派",
      "items": [
        {
          "name": "鹹派雞肉",
          "price": 125,
          "desc": "起司與香草香料"
        },
        {
          "name": "鹹派培根",
          "price": 125,
          "desc": "培根與紅洋蔥"
        }
      ]
    },
    {
      "name": "可頌",
      "items": [
        {
          "name": "原味可頌",
          "price": 70
        },
        {
          "name": "巧克力可頌",
          "price": 80,
          "desc": "比利時黑巧克力"
        }
      ]
    }
  ]
}
```

#### 2.2 轉換為 WooCommerce CSV 格式

**CSV 範例**：
```csv
name,sku,price,description,category,regular_price,stock_quantity
鹹派雞肉,pie-001,125,起司與香草香料,鹹派,125,20
鹹派培根,pie-002,125,培根與紅洋蔥,鹹派,125,20
原味可頌,croissant-001,70,每日新鮮出爐，酥脆層次分明,可頌,70,30
巧克力可頌,croissant-002,80,比利時黑巧克力,可頌,80,25
```

#### 2.3 WooCommerce 批量導入

**步驟**：
1. 後台 → 產品 → 匯入
2. 選擇 CSV 檔案
3. 映射欄位
4. 執行匯入

**需要準備的欄位**：
- SKU（商品代碼）
- 產品名稱
- 描述
- 定期價格
- 分類
- 庫存數量
- 圖片 URL

#### 2.4 添加商品圖片

**圖片要求**：
- 格式：JPG / PNG
- 解析度：2000x2000px（最佳）
- 檔案大小：< 500KB
- 風格：高質感美食照

**圖片來源**：
- 自製食物照片（推薦）
- 專業攝影
- AI 生成圖片

#### 2.5 設定庫存 + 預訂

**每日預訂時間**：
- 鹹派：每日限量 20 個
- 可頌：每日限量 30 個
- 法式甜點：每週限量製作

**庫存設定**：
```
WooCommerce → 產品 → 編輯
→ 庫存選項 → 管理庫存 ✓
→ 庫存數量 20
→ 低庫存閾值 5
```

### ⏳ 第 3 階段：支付 + 運費設置（待開始）

#### 3.1 綠界支付整合

**步驟**（綠界申請完成後）：

1. **安裝綠界外掛**
   ```
   外掛 → 新增 → 搜尋 "ECPay"
   → 安裝 "金流外掛 by ECPAY"
   ```

2. **配置金鑰**
   ```
   WooCommerce 設定 → 支付方式 → ECPay
   商店代號：[申請時取得]
   HashKey：[申請時取得]
   HashIV：[申請時取得]
   測試模式：ON（測試階段）
   ```

3. **測試支付**
   ```
   測試信用卡：4111111111111111
   驗證碼：任意 3 碼
   有效期：未來日期
   ```

#### 3.2 支付方式選項

綠界支援的支付方式：
- 💳 信用卡（一次付款）
- 🏧 ATM 轉帳
- 🏪 超商代碼繳費（7-11、全家、萊爾富、OK）
- 📱 Apple Pay / Google Pay

**建議**：全部啟用（提供客戶更多選擇）

#### 3.3 運費方案

**建議配置**：

| 方案 | 配送方式 | 運費 | 適用範圍 |
|------|----------|------|----------|
| 自取 | 店取 | 免費 | 台北市（指定地點） |
| 宅配 | 黑貓宅急便 | NT$60-120 | 台灣本島 |
| 離島 | 黑貓宅急便 | NT$150+ | 金馬澎 |

**WooCommerce 設定**：
```
WooCommerce 設定 → 運費
→ 分區運費 → 新增運費分區
```
