# claude-practice

練習使用 Claude Code 建立的純前端專案，所有頁面皆為單一 HTML 檔案，用瀏覽器直接開啟即可使用。

---

## 檔案結構

```
claude-practice/
├── index.html         # 練習用首頁（Hello World）
├── accounting.html    # 簡易記帳與損益表工具
├── sales.html         # 鹹派品牌一頁式銷售頁
└── README.md          # 本說明文件
```

---

## index.html　練習用首頁

最初的練習頁，功能簡單，主要用於測試 Claude Code 基本操作。

**功能：**
- 輸入名字，點擊按鈕顯示歡迎訊息

## accounting.html　簡易記帳工具

不需安裝套件，開啟即用的純前端記帳工具。資料存於瀏覽器 `localStorage`，重新整理後不消失。

### 表單輸入

- 欄位：日期、類型、分類、項目名稱、金額、稅額、備註、是否可扣抵
- 類型分為三種：
  - `income`（收入）→ 分類：銷售收入
  - `cost`（成本）→ 分類：進貨材料
  - `expense`（費用）→ 分類：房租、水電、包材、廣告、運費、雜支
- 含稅金額自動計算（未稅 ＋ 稅額），不需手動輸入
- `income` 類型時「是否可扣抵」欄位自動隱藏

### 紀錄列表

- 顯示所有欄位，支援橫向捲動
- 依月份與類型篩選，即時更新
- 每筆紀錄可編輯或刪除
- 編輯時表單自動帶入原始資料，儲存後即時更新

### 損益表摘要（以未稅金額計算）

| 項目 | 說明 |
|------|------|
| 總收入 | 所有 income 紀錄的未稅金額加總 |
| 減：銷貨成本 | 所有 cost 紀錄的未稅金額加總 |
| 毛利 | 總收入 − 銷貨成本 |
| 減：總費用 | 所有 expense 紀錄的未稅金額加總 |
| 淨利（稅前） | 毛利 − 總費用 |

### 營業稅摘要

| 項目 | 說明 |
|------|------|
| 銷項稅額 | income 的稅額合計 |
| 進項稅額合計 | cost ＋ expense 的稅額合計 |
| 可扣抵進項稅額 | deductible = true 的進項稅額 |
| 當期應納／溢付稅額 | 銷項稅額 − 可扣抵進項稅額 |

### 匯出功能

- 可匯出 CSV，支援匯入 Excel 或其他試算表工具

## sales.html　鹹派品牌銷售頁

法式手工鹹派品牌的一頁式銷售頁，目前為**版型稿**，文字內容與店家資訊為示意用途，尚未定稿。

### 頁面結構

| 區塊 | 說明 |
|------|------|
| Navbar | 固定導覽列，含品牌名稱與各區錨點連結 |
| §1 首屏主視覺 | 左側文案 ＋ 右側深棕視覺面板，浮動品牌數據 |
| §2 商品理念 | 品牌故事 ＋ 四項核心理念卡片 |
| §3 商品區 | 六款鹹派卡片，各有獨立色調縮圖與加入購物車按鈕 |
| §4 熱賣推薦 | 單品放大展示，附賣點說明與預訂按鈕 |
| §5 下單方式 | 四步驟流程 ＋ Line / Instagram / Email 聯絡管道 |
| §6 取貨與配送 | 門市自取、低溫宅配、禮盒包裝、食用保存四項說明 |
| §7 FAQ | 六題可展開收合的常見問題 |
| §8 最後 CTA | 深色全幅結尾區塊，含兩個行動按鈕 |

### 設計規格

- **字體**：Playfair Display（標題，含斜體）＋ Noto Sans TC（內文，300 細體）
- **配色方向**：法式雜誌風，暖奶油白 `#F8F4EE` ／ 霧灰米 `#E8DED2` ／ 深可可棕 `#3F2F28` ／ 灰鼠尾草綠 `#A3ACA0`
- **按鈕色**：低飽和金棕 `#B58A69`，hover `#9A6F50`
- **響應式**：820px 以下單欄，600px 以下 Navbar 連結隱藏

### 店家資訊設定

頁面使用 JavaScript 設定檔集中管理店家資訊，只需修改 `sales.html` 底部 `<script>` 內的 `STORE` 物件，即可同步更新全頁所有對應位置：

```js
const STORE = {
  name:         '店名（中文）',
  nameEn:       '店名（英文）',
  tagline:      'Navbar 副標',
  address:      '門市完整地址',
  addressShort: '頁尾簡短地址',
  hours:        '取貨時間',
  line:         'Line 帳號',
  instagram:    'Instagram 帳號',
  email:        '訂購 Email',
  year:         '版權年份',
  footerNote:   '頁尾備註',
};
```
