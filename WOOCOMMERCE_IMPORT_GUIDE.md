## 🔧 問題與解決

### 問題 1：初次導入失敗
**症狀**：匯入 55 商品失敗  
**錯誤訊息**：「不合法的商品類型」

**原因**：CSV 檔案缺少 `type` 欄位，WooCommerce 無法識別商品類型。

**解決**：
- 建立 `nosif_products_v2.csv`
- 為每個商品新增 `type: simple` 欄位
- 重新匯入成功 ✅

---

### 問題 2：可變產品結構
**需求**：同一口味蛋糕應有多個尺寸選項（5吋、6吋、8吋、9吋、10吋）

**方案評估**：
- **方案 A**：使用 WooCommerce 可變產品（Variable Product）
  - 優點：購物體驗更好，選項清晰
  - 缺點：匯入複雜，需要預先定義屬性，CSV 格式難控

- **方案 B**：分兩次匯入 ✅ 採用此方案
  - 第一次：其他商品（簡單商品）
  - 第二次：蛋糕（簡單商品，每個尺寸一個 SKU）
  - 優點：簡單可靠，容易維護
  - 缺點：暫時無法在前端一次選擇尺寸（後續可改進）

## 📤 匯入步驟

### 第一次匯入 — 簡單商品

1. **開啟 WooCommerce 後台**  
   https://nosifpastry.online/wp-admin

2. **進入匯入工具**  
   工具 → 匯入 → 選「WooCommerce」

3. **上傳 CSV**  
   選擇 `nosif_products_simple.csv`

4. **欄位映射**（對照表）

   | CSV 欄位 | 對應 WooCommerce 欄位 |
   |---------|-----------------|
   | name | 名稱 |
   | price | 售價 |
   | description | 簡短的商品說明 |
   | category | 分類目錄 |
   | stock_quantity | 庫存 |
   | regular_price | 原價 |
   | sku | 不要匯入 |

5. **執行匯入**  
   確認後點「匯入」

✅ **結果**：14 個商品成功導入

---

## 🏷️ 分類操作

### 商品分類清單

所有商品導入後都是 **Uncategorized**（未分類），需要手動分配：

| 分類名稱 | 中文 | 包含商品 |
|---------|------|---------|
| 生日蛋糕 | Birthday Cake | 40 個蛋糕 + 名字加購 |
| 可頌 | Croissant | 3 個可頌 |
| 鹹派 | Savory Pie | 2 個鹹派 |
| 麵包 | Bread | 4 個麵包 |
| 法式甜點 | French Dessert | 5 個法式甜點 |

### 分類命名原則

- **名稱**：使用中文（符合品牌和顧客體驗）
- **代稱（Slug）**：英文或留空（WooCommerce 自動生成，用於 URL）

### 批量分類步驟

1. **勾選同一類的商品**  
   例如：所有蛋糕商品

2. **批次操作**  
   上面選「批次操作」→ 選「編輯」

3. **指派分類**  
   找「分類」欄位，勾選目標分類（例如：生日蛋糕）

4. **更新**  
   點「更新」

5. **重複**  
   對其他商品類別重複此過程

## 💡 注意事項

### 關於可變產品
目前蛋糕商品是簡單商品（40 個獨立 SKU），未來如果想改為可變產品（一個蛋糕口味，多個尺寸選項），需要：
1. 在後台定義「尺寸」屬性
2. 創建父商品（例如：「紅心芭樂蛋糕」）
3. 為每個父商品創建 5 個子商品（變體）

這可以在後台手動進行，或使用專門外掛自動轉換。

### 分類管理
- 所有分類都用**中文名稱**（與品牌一致）
- 代稱可自動生成或自訂英文
- 分類可以在「產品 → 分類目錄」統一管理
