# Nosif Pastry 購物車系統 — 實施計劃

**專案名稱**：Nosif Pastry 線上購物

### 階段 2：商品導入與設定（第 1-2 週）

#### 2.1 從菜單生成器導入商品

**步驟 A：匯出菜單資料**
```javascript
// 從編輯器導出為 JSON
{
  "categories": [
    {
      "name": "鹹派",
      "items": [
        {
          "name": "鹹派雞肉",
          "price": 125,
          "desc": "起司與香草香料"
        }
      ]
    }
  ]
}
```

**步驟 B：轉換為 WooCommerce 格式**
- 類別 → WooCommerce 分類
- 商品 → WooCommerce 商品
- 價格、描述、圖片

**步驟 C：批量導入**
```
產品 → 匯入 → 上傳 CSV/JSON → 映射欄位 → 導入
```

#### 2.2 商品信息完善
- [ ] 添加高質感食物照片
- [ ] 補充詳細描述
- [ ] 設定庫存
- [ ] 添加產品標籤（熱銷、推薦、新品）

### 階段 3：主題開發與品牌風格（第 2 週）

#### 3.1 選擇主題
**選項 A：改造現成主題**（推薦，快速）

**字體**

### 階段 4：支付與運費設定（第 2-3 週）

#### 4.1 支付整合

**綠界 ECPay 設定**
```
WooCommerce 設定 → 支付方式 → ECPay
輸入：商店代號、HashKey、HashIV
測試模式 ON
```

**支付流程**
1. 用戶點「前往結帳」
2. 填寫收貨資訊
3. 選擇支付方式（信用卡、ATM、超商付款）
4. 導向綠界支付頁
5. 支付完成 → 訂單確認郵件

#### 4.2 運費設定

**運費方案**
```
| 方案 | 範圍 | 費用 |

| 離島 | 外島 | NT$200+ |
```

## 📊 商品導入範例

### 來源（菜單生成器）
```json
{
  "categories": [
    {
      "name": "鹹派",
      "items": [
        {
          "name": "鹹派雞肉",
          "price": "NT$ 125",
          "desc": "起司與香草香料"
        }
      ]
    }
  ]
}
```

### 轉換為 WooCommerce CSV
```csv
name,sku,price,description,category,image_url,stock
鹹派雞肉,pie-chicken,125,起司與香草香料。內餡選用新鮮法式雞肉,鹹派,https://...,20
```

### WooCommerce 中顯示
- 商品名稱：鹹派雞肉
- SKU：pie-chicken
- 價格：NT$ 125
- 描述：起司與香草香料。內餡選用新鮮法式雞肉
- 分類：鹹派
- 圖片：高質感食物照
- 庫存：20 個

## 📞 常見問題

**Q: WooCommerce 會很慢嗎？**  
A: 不會。使用好的主機（SiteGround、Bluehost）+ 快取外掛就很快。

**Q: 支付安全嗎？**  
A: 綠界是台灣大型支付商，已通過 PCI DSS 認證，很安全。

**Q: 可以同時用菜單生成器和購物網站嗎？**  
A: 可以！菜單生成器用來製作週菜單圖，購物網站用來線上銷售。

**Q: 如何管理商品圖片？**  
A: WooCommerce 有內建圖片管理。建議上傳高質感照片（JPG 2000x2000px）。
